Getting Started

Create an html file. After that, you can start setting up the basic structure of an HTML file.

The basic structure of an HTML file should look like this:

<html>
    <head>
        <title></title>
    </head>
    <body></body>
</html>

Inside the title tag, you can put your desired title for the page, which will appear on the tabs list of your browser.

Linking CSS stylesheets

To add css stylesheets, add this piece of code on your head tag.

<link rel="stylesheet" href="path_to_css.css">

Where path_to_css.css is the name of your css file.

For example, when your css file is named style.css and is stored at the folder css/, the code will look like this:

<link rel="stylesheet" href="css/style.css">

Linking JS files

When you link JS files, they must be linked inside the body. The code snippet will look like this:

<script src="script.js"></script>

For example:

....
<script src="script.js"></script>
</body>
</html>
